Q1 Write the complied CSS from the below Sass/Less expression 

.btn-container {
    .btn {
        color:white;
    }

    &.active {
        .btn {
            color: blue;
        }
    }

    &,
    &:active {
        .btn {
            color: grey;
        }
    }
}
Answer :
.btn-container .btn {
    color: white;
}

.btn-container.active .btn {
    color: blue;
}

.btn-container .btn, .btn-container:active .btn {
    color: grey;
}

Q2 Using Jquery, on click of "btn" add an class "active" to its parent "btn-container" div 


<div class="btn-container"></div>
<div class="btn-container">
    <a class="btn">Button</a>
</div>
Answer :
$('.btn').on('click', function(){
    $(this).parent('.btn-container').addClass('active');
});
